## ⚙️ Lab Highlights

### Lab 04 - Drag Race Traffic Light
Simulated a professional drag race countdown system using 8 LEDs with
500ms timing accuracy and sequential light patterns mimicking a real
Christmas Tree start system.

### Lab 07 - Smart Temperature Alarm
Built a temperature monitoring system using LM35 sensor and LCD display
with a buzzer that triggers automatically when temperature exceeds a
set threshold.

### Lab 09 - Integrated Environmental Monitoring System
The most complex lab - combined LM35 temperature sensing, potentiometer
threshold control, 3x3 LED matrix pattern display, and buzzer alarm into
one cohesive system. If ambient temperature exceeds the user-defined
threshold, the LED matrix displays a warning pattern and the buzzer
activates automatically.
